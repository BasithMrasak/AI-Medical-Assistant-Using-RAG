## Getting Started

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/BasithMrasak/AI-Medical-Assistant-Using-RAG.git
   cd AI-Medical-Assistant-Using-RAG
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
